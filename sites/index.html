My Self-Hosted Sites & Projects

This is a collection of all of the various websites and projects that I am self-hosting. These are not included in my Projects Page, as those are programming projects and these are just apps. Everything you see on this list is hosted on my own hardware, from my apartment. Click a logo to see the live site, and the info box to find the origin of the application! Click me to view a list of all of my hosted sites!

Resume Website

This website is hosted using NodeJS and made with bootstrap. Its main purpose is to host a virtual and interactive resume where a prospective employer can see more about my education or career.

Wedding Website

This website is hosted using a tool called Photo-Stream, which is a tool that I found that can just host multiple images in a minimalistic style and are great for viewing or downloading.

Homepage

This is a self-hosted homepage that I use as my personal homepage, it shows stats, statuses, and I can search using my search engine with it.

Nextcloud

This is a self-hosted file storage application that I use to store and back up files that I deem important to keep around. It is hosted on my own hardware in my apartment.

GitLab

GitLab is a self-hostable source control platform powered by the git framework. I decided to self-host it because I thought it would be cool to show off the code that I write on my own GitLab instance.

Plane

Plane is an Open Source JIRA, Linear and Height Alternative. Plane helps you track your issues, epics, and product roadmaps in the simplest way possible. It can import from GitHub and JIRA.

SearXNG

SearXNG is a self-hostable search engine that aggregates results from various search services and databases. Their main focus is privacy, as no private data is sent to any search services/databases.

Hoppscotch

Hoppscotch is a tool that can send different HTTP requests to urls and endpoints. Think Postman but self-hosted.

Upptime

Upptime is a status page based on GitHub Pages and Actions. It tracks all of my other subdomains and sites, checking their uptime.

Memos

Memos is a quick and lightweight note-taking tool that can be used to saving shower thoughts or for taking notes in class. I installed it because it has been convenient to have a centralized place for notes.

Paste

Paste is a quick and lightweight pastebin-like tool that can be used to share code or text files. I self-hsoted this one so that I don't have to risk any private data being pasted into a site that I do not own.

FilePizza

FilePizza is a quick and lightweight filesharing tool that can be used to share a file with a link similar to Paste or pastebin. This is self-hosted for similar reasons to the paste tool, for privacy.

Docs

This is a simple website where I am hosting all of my documentation (generated or not) for different projects. Things like Javadoc.

AMP (Application Management Panel)

AMP is a game server hosting platform application that makes dedicated game server hosting easy, simple, and containerized. I self host it because it is convenient for hosting servers for others.
