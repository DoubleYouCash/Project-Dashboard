Things I'm Self-Hosting

This is a collection of all of the various websites and projects that I am self-hosting. These are not included in my Projects Page, as those are programming projects and these are just apps. Everything you see on this list is hosted on my own hardware, from my apartment. Click a logo to see the live site, and the info box to find the origin of the application!

Resume Website

This website is hosted using NodeJS and a boostrap template that you can find by clicking on this box. Its main purpose is to host a virtual, extended and interactive resume where a prospective employer or researcher can see more information about my education or career.

Wedding Website

This website is hosted using a tool called Photo-Stream, which is a tool that I found that can just host multiple images in a minimalistic style and are great for viewing or downloading. Credit goes to waschinski on GitHub.

Nextcloud

This is a self-hosted file storage application that I use to store and back up files that I deem important to keep around. It runs using PHP and is entirely open source, hosted on my own hardware in my apartment.

GitLab

GitLab is a self-hostable source control platform powered by the git framework. I decided to self-host it because I thought it would be cool to show off the code that I write on a GitLab instance that I self host. I could also host friends code on there if the need arose.

Upptime

Upptime is a status page based on GitHub Pages and Actions. It tracks all of my other subdomains and sites, checking their uptime.

Memos

Memos is a quick and lightweight note-taking tool that can be used to saving shower thoughts or for taking notes in class. I installed it because it has been convenient to have a centralized place for notes not named OneNote.

Paste

Paste is a quick and lightweight pastebin-like tool that can be used to share code or text files. I self-hsoted this one so that I don't have to risk any private data being pasted into a site that I do not own.

FilePizza

FilePizza is a quick and lightweight filesharing tool that can be used to share a file with a link similar to Paste or pastebin. This is self-hosted for similar reasons to the paste tool, for privacy.

Nginx Proxy Manager

Nginx Proxy Manager is a web client that allows you to control all of your nginx reverse proxies in one place, which is useful for me because you can see that I have a lost of sites that I host on one machine.

Homer

Homer is a dashboard tool that can show all sites being hosted in any homelab or other setup. It's sort of what this site is, but organized seperately, easier to share, and has less info.

AMP (Application Management Panel)

AMP is a game server hosting platform application that makes dedicated game server hosting easy, simple, and containerized. I self host it because it is convenient for my own personal game servers and I host servers for others.
