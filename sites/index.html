Things I'm Self-Hosting

This is a collection of all of the various websites and projects that I am self-hosting. These are not included in my Projects Page, as those are programming projects and these are just apps. Everything you see on this list is hosted on my own hardware, from my apartment. Click a logo to see the live site, and the info box to find the origin of the application!

Wedding Website

This website is hosted using a tool called Photo-Stream, which is a tool that I found that can just host a bunch of images in a minimalistic style and are great for viewing or downloading. Credit goes to waschinski on GitHub.

Nextcloud

This is a self-hosted file storage application that I use to store and back up files that I deem important to keep around. It runs using PHP and is entirely open source, hosted on my own hardware in my apartment.

GitLab

GitLab is a self-hostable source control platform powered by the git framework. I decided to self-host it because I thought it would be cool to show off the code that I write on a GitLab instance that I self host. I could also host friends code on there if the need arose.

Bitwarden - Not Currently Working

Bitwarden is a self-hostable password manager that can generate passwords and store saved passwords with encryption to ensure security. It is not currently working.

AMP (Application Management Panel)

AMP is a game server hosting platform application that makes dedicated game server hosting easy, simple, and containerized. I self host it because it is really nice for my own personal game servers and I host servers for others.

Memos

Memos is a quick and lightweight note-taking tool that can be used to saving shower thoughts or for taking notes in class. I installed it because I thought it would be really nice to have a centralized place for notes not named OneNote.
